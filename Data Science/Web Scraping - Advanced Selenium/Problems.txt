
1. Why do we need to use waits?
Answer: Page is not loaded yet, Element to interact with is not available in DOM yet, Element to interact with is not clickable yet, AJAX is still loading and element will be created 
after AJAX 


2. Which of the following is true about implicit wait?
Answer: Tells WebDriver to wait for a certain amount of time when trying to find an element, If we provide wait time as 20 secs, and our element is being loaded at 5 sec, but still 
it will wait for 20 sec, Implicit wait is applied globally


3. Which of the following statements are correct?

4. Which of the following are the available conditions we can use in explicit wait?
Answer: invisibility_of_element_located, element_to_be_selected, frame_to_be_available_and_switch_to_it, element_located_selection_state_to_be


5. Which of the following condition allows waiting for an element to find with a given text inside the locator?
Answer: text_to_be_present_in_element_value


6. Answer the following question by looking at the given line of code :- wait = WebDriverWait(driver, 5, pollfrequency = 1, ignoredexceptions = None)
Answer: Number of seconds before timing out is 5s, sleep interval between calls 1s


7. Answer the following question by looking at the given line of code :- wait = WebDriverWait(driver, 5, pollfrequency = 1, ignoredexceptions = None)
Which of the following completes the code if we want to wait for an element with id = “cn” to till it is in a selectable state?
Answer: wait.until(EC.element_to_be_selectable((By.ID, ”cn”)))

19. A page shows a list of products.If it shows 24 entries initially and In each scrolling of page, 15 products are added. If I scroll the page 27 times,how many products will I be 
able to scrape?
Answer: Number of products scraped in 27 scrolls = 405. Number of products initially available = 24. Total Number of products scraped = 405 + 24 = 429
